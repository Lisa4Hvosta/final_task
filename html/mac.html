Терминал MacOS

Навигация по файловой системе

  • ls – Показать содержимое текущего каталога
  • cd путь – Перейти в указанный каталог
  • pwd – Показать текущий рабочий каталог
  • find путь -name "имя" – Найти файлы по имени

Работа с файлами и папками

  • mkdir имя – Создать новую папку
  • cp источник цель – Копировать файлы
  • mv источник цель – Переместить или переименовать файлы
  • rm файл – Удалить файл
  • rm -r папка – Удалить папку рекурсивно

Просмотр и редактирование

  • cat файл – Показать содержимое файла
  • nano файл – Открыть файл в текстовом редакторе nano
  • grep "текст" файл – Найти текст в файле
  • chmod права файл – Изменить права доступа к файлу
