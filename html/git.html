Git

  • git log --oneline - посмотреть все коммиты.
    • git checkout - восстановить все. git checkout "код коммита" - вернуть до состояния этого коммита. git checkout master - вернуться в ветку мастер. git fetch --all - Восстановить файлы на локальном компьютере git reset --hard origin/master или git reset --hard origin/<название_ветки> git add text.txt - Добавить файл в репозиторий git rm text.txt - Удалить файл git status - Текущее состояние репозитория (изменения, неразрешенные конфликты и тп) git commit -a -m "Commit description" - Сделать коммит git push origin - Замерджить все ветки локального репозитория на удаленный репозиторий
