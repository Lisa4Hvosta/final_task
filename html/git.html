Команды Git

  • git init — инициализация нового репозитория.
  • git add . — добавление изменений в индекс (подготовка к коммиту).
  • git commit -m "message" — создание коммита с описанием изменений.
  • git branch — просмотр и управление ветками.
  • git checkout branch_name — переключение между ветками.
  • git push — отправка изменений на удалённый репозиторий.
  • git pull — получение изменений из удалённого репозитория.
