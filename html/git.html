Команды Git

  • git init: Создает новый пустой локальный репозиторий.
  • git clone [url]: Клонирует (копирует) существующий удаленный репозиторий на локальную машину.
  • git add [файл]: Добавляет изменения в файл(ы) в индекс (staging area) для подготовки к коммиту.
  • git commit -m "[сообщение]": Фиксирует изменения в репозитории с комментарием, сохраняя историю.
  • git status: Показывает состояние рабочего каталога: какие файлы изменены, добавлены или готовы к коммиту.
  • git push: Отправляет зафиксированные локальные коммиты на удаленный репозиторий.
  • git pull: Получает изменения из удаленного репозитория и объединяет их с текущей локальной веткой.
  • git branch: Показывает список веток, позволяет создавать или удалять их.
  • git checkout [ветка]: Переключает на указанную ветку (или git switch).
  • git merge [ветка]: Объединяет указанную ветку с текущей рабочей веткой.
  • git log: Отображает историю коммитов (журнал) для текущей ветки
