Git

Основные команды Git

  • git init – Инициализация нового репозитория
  • git add . – Добавить все файлы в индекс (staging area)
  • git commit -m "сообщение" – Создать коммит с описанием
  • git status – Показать состояние рабочего каталога и индекса
  • git log – Показать историю коммитов

Работа с удаленными репозиториями

  • git clone url – Клонировать удаленный репозиторий
  • git push – Отправить изменения в удаленный репозиторий
  • git pull – Забрать изменения из удаленного репозитория
  • git remote add origin url – Добавить удаленный репозиторий

Ветвление и слияние

  • git branch – Показать список веток
  • git checkout -b name – Создать и перейти в новую ветку
  • git merge branch – Слить указанную ветку с текущей
  • git diff – Показать изменения между коммитами
