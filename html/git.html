Git

  • git init - инициализация (initialisation) репозитория (создание репозитория).
  • git clone - клонирование удаленного репозитория с GitHub на локальный компьютер.
  • git push - внесение изменений на удаленный репозиторий.
  • git branch - создание ветки в локальном репозитории.
