Командная строка Windows

Управление файлами и папками

  • COPY — Копирует файлы в другое место
  • DIR — Отображает файлы и папки в текущем каталоге
  • CD — Изменяет текущий каталог
  • MAKEDIR (или MD) — Создаёт новую папку
  • MOVE — Перемещает файлы или папки в другое место

Приложения и процессы

  • TASKLIST — Показывает список выполняемых процессов
  • TASKKILL — Завершает выполнение процесса (для остановки используется PID, который можно узнать через TASKLIST)

Системная информация

  • DATE — Показывает или устанавливает текущую дату
  • TIME — Показывает или устанавливает системное время
  • HOSTNAME — Показывает имя компьютера
  • SYSTEMINFO — Показывает информацию о конфигурации системы
  • VER — Показывает версию Windows
