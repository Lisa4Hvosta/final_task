Git

  • git init - Создаёт новый Git-репозиторий в текущей папке.
  • git clone - Скачивает удалённый репозиторий (например, с GitHub) на компьютер.
  • git status - Расшифровывается как "status" — «статус». Показывает, какие файлы изменены, какие добавлены, а какие ещё не отслеживаются.
  • git add - Добавляет файлы в область подготовки (staging area) перед коммитом.
  • git commit - Сохраняет изменения в истории проекта.
  • git log - Показывает историю коммитов.
