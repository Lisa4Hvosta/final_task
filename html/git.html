Git

  • git init - инициализация нового Git-репозитория в текущей папке.
  • git add . - добавление всех изменений в индекс (staging).
  • git commit -m "сообщение" - создание коммита с описанием изменений.
  • git status - показывает статус файлов (изменённые, добавленные и т.д.).
  • git log - показывает историю коммитов.
  • git branch - показывает список веток или создаёт новую.
  • git checkout - переключение между ветками или восстановление файлов.
  • git push - отправка коммитов в удалённый репозиторий.
  • git pull - получение изменений из удалённого репозитория.
  • git clone - клонирование репозитория.
